# 0x01 Docker 环境判断

1. 查找根目录下是否有****.dockerenv文件****
    
    ```bash
    ls -lah /
    ```
    
2. 查询/proc/1/cgroup是否存在docker字符串
    
    ```bash
    cat /proc/1/cgroup
    ```
    
3. 查看docker容器的环境变量
    
    ```bash
    cat /proc/1/environ
    ```
